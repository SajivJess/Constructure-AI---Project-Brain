# Sample Construction Documents

This folder should contain sample construction project files for testing.

## Expected Documents

For full testing, upload PDFs containing:

1. **Specifications Document** (specs.pdf)
   - Construction specifications
   - Material requirements
   - Fire ratings
   - Accessibility requirements

2. **Drawing Set** (drawings.pdf)
   - Floor plans
   - Door schedules
   - Room layouts
   - Dimensions

3. **MEP Schedule** (mep.pdf)
   - HVAC systems
   - Electrical equipment
   - Plumbing fixtures

## Creating Test Documents

If you don't have real construction documents, you can:

1. Use sample construction PDFs from online resources
2. Create a test PDF with construction-related text
3. Use any technical specification document

## Upload Instructions

1. Start the application
2. Login with demo credentials
3. Go to Documents tab
4. Upload PDF files
5. Wait for processing (shows chunk count)
6. Go to Chat tab and start asking questions

## Sample Queries

After uploading documents, try:
- "What is the fire rating for corridor partitions?"
- "Generate a door schedule"
- "List all rooms with their specifications"
- "What are the HVAC system requirements?"
- "Are there accessibility requirements for doors?"
